# goit-nodejs-hw-01

## About

Basic CLI application to read and write contacts list and make simple actions on them.</br>

### Actions results:

#### node index.js --action list

![list](/images/HW-01-NODEJS-1.jpg "List")

#### node index.js --action get --id <ID_contact>

![get](/images/HW-01-NODEJS-2.jpg "Get")

#### node index.js --action add --name "Mango" --email "mango@gmail.com" --phone "322-22-22"

![add](/images/HW-01-NODEJS-3.jpg "Add")

#### node index.js --action remove --id <ID_contact>

![remove](/images/HW-01-NODEJS-4.jpg "Remove")
